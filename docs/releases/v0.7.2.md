# HandBrake Completed Manager 0.7.2

Version 0.7.2 contains the same-extension replacement and first-paint progress fixes from 0.7.1, with corrected confirmation, progress, and completion wording for atomic in-place replacements.

## Fixes

- Same-extension output can atomically replace the original source path after the converted copy and original backup are SHA-256 verified.
- The confirmation correctly explains that an in-place replacement retains the verified original backup rather than moving the original path to the Recycle Bin.
- Progress accurately reports atomic replacement verification instead of displaying an inapplicable Recycle Bin stage.
- Replacement and bulk-history operation windows render fully before work begins.
