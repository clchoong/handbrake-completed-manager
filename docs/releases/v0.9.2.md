# HandBrake Completed Manager 0.9.2

Version 0.9.2 corrects completed-history alignment and keeps the Record Details area stable across selection changes.

## Completed-history presentation

- Data text now receives the 16-by-6 inset directly, matching each column header's title padding.
- The number cell no longer overrides selection with orange; it inherits the same blue selection background as the other cells and keeps white numbering text.

## Stable Record Details area

- Record Details permanently reserves a fixed 160-pixel area.
- No selection displays guidance to select one item.
- Multiple selections display the selected count and: “Multiple items selected. Select one item to view its record details.”
- Single selection displays the existing paths, status, metrics, and availability details.
- Switching between zero, one, and multiple selections no longer changes the completed-library height.

## Compatibility and validation

The package supports 64-bit Windows 10 build 17763 or later and Windows 11. The Release solution builds with zero warnings and errors, and all 242 automated tests pass.
