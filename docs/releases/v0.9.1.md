# HandBrake Completed Manager 0.9.1

Version 0.9.1 refines the completed-history table and makes aggregate bulk replacement progress continuous.

## Table interaction and styling

- Column headers and data cells use matching 16-by-6 padding.
- The selected number cell uses the Replace Source orange with white text.
- Output percentages below 80% retain normal table styling.
- Values from 80% through 89% use the solid Replace Source orange with white text.
- Values from 90% use the equivalent solid red treatment with white text.
- The percentage treatment remains the same when its row is selected or highlighted.
- Pressing **Delete** while the table is focused opens the existing confirmed Remove History workflow for the current selection; media files are never deleted by this shortcut.

## Real-time bulk progress

- The overall progress bar now includes the active item's live byte progress instead of moving only after an item finishes.
- In a five-item batch, 50% progress through item one displays 10% overall; completing item one displays 20% and `1/5`.
- Individual progress bars, cancellation, stop-after-current, and visible per-item outcomes remain unchanged.

## Compatibility and validation

The package supports 64-bit Windows 10 build 17763 or later and Windows 11. The Release solution builds with zero warnings and errors, and all 242 automated tests pass.
