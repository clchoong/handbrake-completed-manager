# HandBrake Completed Manager 0.8.1

Version 0.8.1 is a maintenance release for completed-history interaction and direct bulk replacement.

## Fixes and improvements

- Double-click opens the effective playable file. It uses the replacement file after the default replacement has consumed the separate output.
- Bulk source replacement advances through all eligible records after the initial bulk confirmation; individual successful, cancelled, or failed steps no longer wait for **Done**.
- Automatic history checks no longer rebuild an unchanged list every few seconds. When receiver data has changed, selected record IDs are restored after the update.
- History text is vertically centred and column-title dividers improve scanning.
- Remove from History also removes internal legacy recovery rows belonging to the selected record, fixing foreign-key failures without changing any media file.

## Compatibility

The package supports 64-bit Windows 10 build 17763 or later and Windows 11. Existing portable databases upgrade in place without losing completed history.

## Validation

The Release build completes with zero warnings and zero errors. All 229 automated tests pass, including regression coverage for removing a history record that has legacy replacement metadata.
