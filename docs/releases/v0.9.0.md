# HandBrake Completed Manager 0.9.0

Version 0.9.0 improves dense-history readability and makes multi-file source replacement progress visible in one place.

## Completed-history table

- Cells have additional horizontal and vertical padding.
- A 1-based number column follows the current visible sort and filter order.
- Output percentages above 80% use pale orange highlighting.
- Output percentages of 90% or more use pale red highlighting.

## Unified bulk replacement progress

- One persistent window lists every eligible replacement item.
- Every item has an individual byte-progress bar and current status.
- An item-based overall bar and `processed/total` counter update after each attempted replacement. A five-item batch therefore moves from `0/5` to `1/5` and 20% after its first item.
- Completed, failed, cancelled, waiting, and skipped outcomes remain visible until the user closes the finished summary.
- **Stop after current** remains available, and **Cancel current & stop** is enabled while the current copy is cancellable.

## Compatibility and validation

The package supports 64-bit Windows 10 build 17763 or later and Windows 11. The Release solution builds with zero warnings and errors, and all 240 automated tests pass, including the output-percentage boundaries and `0/5` through `5/5` bulk-progress calculations.
