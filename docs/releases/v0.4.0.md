# HandBrake Completed Manager 0.4.0

This is the first public open-source release of HandBrake Completed Manager, an independent Windows companion that preserves source-to-output history after a HandBrake encode completes.

> **Independent third-party project:** This project is not affiliated with, endorsed by, or maintained by the HandBrake project. “HandBrake” is used only to identify compatibility with the official [HandBrake](https://handbrake.fr/) application. The package does not include HandBrake code, binaries, documentation, or artwork.

## Highlights

- Durable SQLite history populated by HandBrake's configurable completion action
- Search, filters, size comparisons, playback, and Explorer file location
- One-click safe source replacement with preflight checks, verified copies, durable recovery checkpoints, Windows Recycle Bin retirement, and undo
- Verified **Recycle output** for encodes that fail manual quality review
- Source-replacement status shown directly in completed history
- High-contrast original application icon and refined Windows desktop interface
- Single-instance behavior that prevents duplicate notification-area icons
- Mouse-wheel handoff between the history table and the surrounding page
- Windows 10 build 17763+ application target and Windows 11 support

## Download

Download `HandBrake-Completed-Manager-0.4.0-win-x64.zip`, extract the complete folder, and read `PORTABLE-README.txt` before configuring HandBrake.

The package is self-contained for 64-bit Windows and requires neither administrator access nor a separate .NET installation. It is not code-signed, so Windows may display a reputation warning.
