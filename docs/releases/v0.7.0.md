# HandBrake Completed Manager 0.7.0

Version 0.7.0 simplifies the everyday replacement workflow, makes bulk history removal visibly responsive, fixes Settings accessibility, and adds complete in-app software information.

> **Independent third-party project:** This project is not affiliated with, endorsed by, or maintained by the HandBrake project. “HandBrake” is used only to identify compatibility with the official [HandBrake](https://handbrake.fr/) application. The package does not include HandBrake code, binaries, documentation, or artwork.

## Highlights

- Replace a source through one exact-path warning and one **Replace source** button.
- Follow one familiar overall progress bar while copy, backup, verification, installation, Recycle Bin retirement, and transaction completion run internally.
- Keep detailed stage controls out of the normal workflow and available through Recovery only when an interrupted checkpoint needs attention.
- Close the bulk-removal confirmation before database work begins, then show a dedicated responsive progress window with the current item and final totals.
- Scroll the Settings content independently of its fixed Save/Cancel footer so Diagnostic logs remains reachable at smaller window sizes and higher display scaling.
- Open a new About window for the real packaged version, MIT licence, copyright, independence notice, Windows support, .NET runtime, architecture, portable distribution, local-storage boundary, project page, and licence notices.
- Publish using isolated single-process build servers to prevent stalled parallel packaging jobs on Windows.

The internal replacement safety model is unchanged: a converted copy and original backup are verified before the source is moved to the Windows Recycle Bin, no unrelated file is overwritten, and durable recovery checkpoints remain available.

## Download

Download `HandBrake-Completed-Manager-0.7.0-win-x64.zip`, extract the complete folder, and read `PORTABLE-README.txt` before configuring HandBrake.

The package is self-contained for 64-bit Windows and requires neither administrator access nor a separate .NET installation. It targets Windows 10 build 17763 or later and Windows 11. It is not code-signed, so Windows may display a reputation warning.
