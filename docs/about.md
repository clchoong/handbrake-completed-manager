# About and software information

The dashboard header provides an **About** action. The About window identifies:

- The packaged application version read from the executing assembly.
- The MIT project licence and copyright holder.
- The independent-project and HandBrake compatibility notice.
- Supported Windows versions, .NET runtime, process architecture, portable distribution, and local-storage boundary.
- The public project page and the packaged `LICENSE.txt` file or repository licence fallback.

The About window does not read machine-local project metadata or contact any service automatically. External links open only when the user selects them. Distributed dependency notices remain in `THIRD-PARTY-NOTICES.txt` and `DOTNET-THIRD-PARTY-NOTICES.txt` beside the executable.
