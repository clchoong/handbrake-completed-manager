# Support

Use this repository's issue forms for reproducible HandBrake Completed Manager bugs and feature requests. Before posting, remove private paths, filenames, media, credentials, and personal database contents.

This project cannot provide support for HandBrake encoding behavior, presets, codecs, or its Queue. For those questions, use the official [HandBrake documentation](https://handbrake.fr/docs/) and [community support](https://handbrake.fr/community.php).

HandBrake Completed Manager is a volunteer-maintained independent project and does not provide guaranteed response times or commercial support.
